## Modes

| key | value |
|---|---|
| git_mode_status_deductions | off |

`git_mode_status_deductions: off` — payloads generated by `scripts/extract_payload.py` come from raw git history, which carries no `[Approved]/[Draft]/[Archived]` lifecycle labels. In git mode the status-label deductions are skipped (signature = commit author, timestamp = commit timestamp, path = stable identifier). Hand-assembled payloads are unaffected. Set `on` only if your repo encodes lifecycle labels in commit messages.
