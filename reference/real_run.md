# REAL RUN — gating a published repository

Everything in `examples.md` uses a fictional project. This run does not. The target is **beacon-hunter** (github.com/mrcord77/beacon-hunter), a public network-security research release (AGPL-3.0, Zenodo DOI 10.5281/zenodo.20431555). The payload below was generated by `scripts/extract_payload.py` from the repository's actual HEAD commit; the intent file was drafted strictly from the repo's own public README and release history. Anyone can clone the repo and reproduce this run.
